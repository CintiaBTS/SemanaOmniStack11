Semana Omnistack 11 da Rocketseat
Projeto: Be The Hero

Semana Omnistack 11 da Rocketseat, no qual será desenvolvido uma aplicação utilizando NodeJs, React e React Native

Neste projeto foi desenvolvido uma aplicação para ONGs de animais, onde será possivel cadastrar animais feridos que necessitam de tratamento, além dos contatos pelo Whatsapp e email (para pagar pelo tratamento do animal).

Aprendizado:
Backend
Neste projeto foi utilizado o knex para configuração do nosso banco de dados, sendo usado o sqlite para development.
Insomnia para testar as rotas
